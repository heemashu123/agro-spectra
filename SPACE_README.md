# Agro-Spectra

> **Meta PyTorch & OpenEnv Hackathon Submission**

A PPO reinforcement learning agent that learns to optimally irrigate and fertilize
a simulated farm across a 30-day crop cycle -- maximizing yield while avoiding chemical runoff.

See the full project on [GitHub](https://github.com/heemashu123/agro-spectra).
